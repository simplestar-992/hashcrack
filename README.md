# HASHCRACK

**Multi-Algorithm Hash Cracker**

## Features

- 🚀 Fast and efficient
- 📦 Single binary, no dependencies  
- 🔧 Easy to use
- 🔒 Secure by design

## Installation

```bash
git clone https://github.com/simplestar-992/hashcrack.git
cd hashcrack
go build -o hashcrack .
```

## Usage

```bash
./hashcrack --help
```
